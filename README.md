# MortgageVision 📊

Android-приложение для визуализации и управления ипотечными данными.

---

## 📱 Описание

MortgageVision — учебное Android-приложение, созданное в Android Studio.  
Проект демонстрирует работу с интерфейсом, навигацией и структурой Android-приложения.

---

## ⚙️ Функционал

- Главный экран приложения
- Навигация между экранами
- Отображение UI элементов
- Базовая логика приложения

---

## 🧱 Технологии

- Kotlin / Java
- Android Studio
- XML (разметка интерфейса)
- Git & GitHub

---

## 📁 Структура проекта

app/
├── java/ — логика приложения  
├── res/layout/ — экраны (UI)  
├── res/drawable/ — изображения  
├── AndroidManifest.xml
